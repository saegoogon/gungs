# WiFi Spot

집 안 와이파이 환경을 쉽게 살펴볼 수 있는 한국어 웹사이트입니다.

## 포함된 내용

- 와이파이 상태를 보여주는 메인 히어로 섹션
- 집 크기, 연결 기기 수, 벽 개수로 확인하는 간단한 신호 진단
- 2.4GHz, 5GHz, 듀얼 밴드 선택에 따른 추천 문구
- 공유기 배치 예시와 설치 팁

## 실행

```bash
npm start
```

브라우저에서 `http://localhost:3000` 으로 열면 됩니다.

## 앱 이름 변경

환경 변수 `APP_NAME` 을 설정하면 사이트 이름을 바꿀 수 있습니다.
